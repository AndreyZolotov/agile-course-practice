## Задание

Разработать приложение для конвертации длин. Должна быть релизована поддержка
работы со следующими единицами измерения:

 - Метр
 - Километр
 - Миля
 - Дюйм

Все необходимые сведения об указанных длинах можно найти на сайте Wikipedia:
["Длина"][Length].
