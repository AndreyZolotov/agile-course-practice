# Очередь

Выполнила:

 - Бородулина Любовь
 - ННГУ, ф-т ВМК, каф. МО ЭВМ, группа 85м21

## Задание

Реализовать класс очереди, имеющий следующие методы:

 - Добавление элемента
 - Удаление элемента
 - Чтение "первого" элемента
 - Проверка на пустоту
 - Очистить очередь

Все необходимые сведения можно найти в статье
["Очередь"][queue] на сайте Wikipedia.
